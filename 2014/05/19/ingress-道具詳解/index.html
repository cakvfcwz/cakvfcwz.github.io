生產

Resonator

resonator 是這遊戲的必須道具，可以沒有武器但不能沒有 resonator.

八級（及以上） Agent 的部署限制是一級八箇、二至四級四箇、五級和六級兩箇、七級和八級一箇，最大能獨立製造 5.625 級 portal. 這一級別的 portal 已經足以應付 regional 級別的 Link. 因此為了做 Field 必須能保持一定的獨立製造四級以上 Portal 的能力，而由於限制，如果 Agent 只有高級的 resonator 那麼他只能製造一級 portal （或者當場 hack 出六根一級 resonator 的話，二級）。一級 portal 的 Link 距離只有 160m, 遠遠不夠用。

考慮到其他裝備的存放，一位八級 Agent 想要插滿一百 portal 的話就需要七級和八級各一百、五級和六級各兩百、一至四級共兩百，這樣就已經八百了。如果還想隨手佔領 portal 而不想或沒時間 Link, 則還需要額外的一兩百高級 resonator. 基本上綜合型 Agent 的裝備裡會有一半左右是等級各異的 resonator.

Heat Sink 和 Multi Hack

Heat Sink 和 Multi Hack 主要用於 Farm 區，一般 Common 級就足夠了，對於在室內不動也能長時間接觸的 portal 在被攻擊不頻繁的時候則可以使用更高級別的。

此外，在外做 Field 卻總是 Hack 不出 Key 的時候也可以臨時使用。

對於生產型 Agent, 這兩種 Mod 的存量應該各有 50 以上，而且 Very Rare 級的不少於 10 箇。綜合型的話，則各有 20 左右就可以滿足了。

這兩種道具的 Very Rare 級別比較稀有，不一定要長期保持標準量。

Link Amp

除了做省際甚至國際的大型 Link 和給隊友搗亂以外沒有任何用途的垃圾，除了出差非常頻繁的 Agent 以外任何 Agent 手中的存量都不應當超過 10 箇。

Portal Key

Portal Key 在做 Field 的時候是戰略資源，在做 Recharge 和 Guardian 成就的時候是必備物品，在收集情報的時候是可選項，其他時候只是佔用位置的垃圾而已。

因此，當自己的長期活動範圍周邊有敵方駐地的時候，可以保有一套 Key, 每當有隊友去收割 AP 就可以相互提供；如果附近己方陣營成員基本只有自己，那麼也可以保有一些 Key 以期待外出時與人交換和做出大型 Field。如果並非以上情況，則這些 Key 通常是隨地亂扔的。

Key 的交換是重要活動，假如有熟悉的隊友各自將他人駐地的 Key 保有許多，則中間地區幾乎可以保證無間歇覆蓋。

但是距離太遠的 Key 是沒有用的，在 Portal 密集的地方，距離限制還會大大縮短。

武器

XMP

XMP 是主要的武器。攻擊型的 Agent 將至少長期保持 1000 以上的頂級 XMP, 我所謂頂級是就該 Agent 的當前級別而言。而綜合型 Agent 則一般保有 200 左右的頂級 XMP 和大致相同數量的低於自己級別兩級以內的 XMP 即可。

全部使用頂級 XMP 是不明智的，低級 XMP 更容易獲得，使用消耗的 XM 也更少。通常最低級別的 XMP 用於攻擊低級 portal, 而低一級的 XMP 則用於低於自己等級一級以內的、Mod配置不齊全的 Portal, 頂級 XMP 一般只用於滿狀態同級以上 portal.

一般如此配置的六級 Agent 面對配置不全的高級 portal 就已經可以比較輕鬆地打下來了。

Ultra Strike

Ultra Strike 以前曾被判定為雞肋，由於新的破盾功能，這一評價必須得到修正。

Ultra Strike 的攻擊範圍極小，使用時必須站在 resonator 的位置才有效，雖然稍微偏一點也可以。對無 Shield 目標，非暴擊狀態下低一兩級的 Ultra Strike 已經可以在中心附近造成 15% 左右的傷害，而較邊緣也有 7%, 而同級別的 Ultra Strike 則幾乎可以秒殺。

站在 portal 中心，則有高幾率破壞 Shield. 基本的使用方法就是先破盾，再根據 resonator 的位置選擇武器，一般會使用 XMP.

當然在團戰的時候， Ultra Strike 除了專用的破盾手以外沒什麼用處。日常作戰中，即使是攻擊型 Agent 配備一百以下的頂級 Ultra Strike 就已經足夠了。

Jarvis Virus 和 ADA Refactor

一般稱之為綠毒和藍毒，異色病毒主要用於解決本陣營 Link 的阻礙（敵方陣營的直接上 XMP 打掉就好了），所以通常異色病毒的戰略地位大於己方陣營病毒。

對於非常高級的 portal 而又必須打掉，低級別的 Agent 可以使用病毒，但會消耗大量 XM. 使用前必須確認自己不會浪費掉。

防禦

Shield

Common Shield 非常容易獲得，按照現行的規則 Common Shield 的防禦是 30, 放滿就是 120. 面對防禦 120 的 5.625 級 portal 八級 Agent 如果不用 Ulta Strike 也比較吃力。

Rare Shield 也不算很稀有，對於 Farm 區的節點，可以使用兩箇 Rare Shield. 團戰當然是全部放 Very Rare 級了。

未證實的說法中，Shield 方位不同，從第一箇槽依次是東北西南。

Force Amp 和 Turret

配合使用，提高 portal 的攻擊力。將敵方 Agent 的 XM 消耗完畢則自然解除危險。

單獨使用一種會讓效果大大降低。

由於每箇人只能安放兩箇 Mod 且 Shield 比較容易獲得， 一般還是首先放置 Shield. 一般非 Farm 用 portal 的標準配置是 Force Amp、Turret 各一箇加上 Shield 兩箇，Shield 的等級由 portal 的位置決定。

其他物品

Power Cube

高級 Power Cube 的主戰場是團戰，而低級 Power Cube 則是一人拆 Field.

在密集的、Mod 配置標準的八級 portal 中釋放一次 XMP, 八級 Agent 會受到非暴擊 50% 的傷害，七級 Agent 如果非滿 XM 則幾乎是秒殺。應當留下一點 XM 釋放甩冬波，故意被防禦反擊打成 0 以後使用低級 Power Cube 繼續打，這是節省 XM 的辦法。

一般對於綜合型 Agent 來說，保持頂級 Power Cube 三十左右，低級的十多箇就足夠了。當然攻擊型則需要一百以上，團戰則需要二百到五百的頂級 Power Cube.

Capsule

一般有一兩箇就足夠了，多了也沒什麼用，除非想做軍火生意。

主要用於倒裝備，相互交換裝備的時候比較方便，當然人員狀況複雜的時候不能用。

做 Field 需要 Key 的時候，可以存放到 Capsule 裡面，效果等同於扔掉，只不過這樣更安全。

Media

除了解密者以外只能回收掉的東西，普通 Agent 沒有關注的必要，到手即可回收。
